クイックソート

クイックソートは、高速なソートアルゴリズムの一つです。分割統治法に基づいており、以下の手順で動作します。

  1. 配列からピボットを選択する。
  2. ピボットより小さい要素と大きい要素に分割する。
  3. 再帰的に各部分をソートする。

クイックソートの平均計算量はO(n log n)ですが、最悪の場合はO(n^2)となることがあります。

実装例

Lumote - 擬似コード


function quickSort(array)
    if length(array) < 2
        return array
    pivot = array[length(array) / 2]
    left = []
    right = []
    for i in 0 to length(array) - 1
        if array[i] < pivot
            left.push(array[i])
        else
            right.push(array[i])
    return quickSort(left) + [pivot] + quickSort(right)

Hoare - 擬似コード


function quickSort(array, low, high)
    if low < high
        pivotIndex = partition(array, low, high)
        quickSort(array, low, pivotIndex - 1)
        quickSort(array, pivotIndex + 1, high)
    return array

function partition(array, low, high)
    pivot = array[high]
    i = low - 1
    for j in low to high - 1
        if array[j] < pivot
            i = i + 1
            swap(array[i], array[j])
    swap(array[i + 1], array[high])
    return i + 1

クイックソートは、効率的で広く使用されているソートアルゴリズムです。適切なピボットの選択と実装により、非常に高速なソートが可能です。
