cjpegli コマンド備忘録

基本的な使い方

cjpegli input output.jpg [options]

基本的なオプション:

  • -q: 品質を指定 (0-100, 100が最良 よくあるJPEGエンコーダーとかの品質と似ているスケール)
  • -d: 品質を指定 (0.0-25.0, 0が最良 butteraugliという指標 距離で示す。 1がニアロスレス)
  • -p: プログレッシブレベル (0-2, 0がベースライン, 2がデフォルト)
  • --xyb: xybカラースペースを使用
  • --chroma_subsampling: クロマサブサンプリングを指定 (444,440,422,420)
  • -v: 詳細な出力を表示
  • -h: ヘルプを表示

例:


                cjpegli input.png output.jpg -q 75 -p 2 --xyb --chroma_subsampling 420
                // これは、品質75、プログレッシブレベル2、xybカラースペース、420クロマサブサンプリングでJPEGを生成します。
            

                cjpegli input.jpg output.jpg -d 1.0 -p 1
                // これは、品質1.0、プログレッシブレベル1でJPEGを生成します。

リポジトリ

cjpegliのソースコードはGitHubのlibjxlリポジトリで公開されています。

謝辞

cjpegliは、Googleのlibjxlプロジェクトの一部であり、JPEG XLエンコーディングのためのコマンドラインツールから派生したツールです。開発者の皆様に感謝します。
